JARVIS

[ ASCII · SHAPES · 3D TEXT · FUN ART ]

A fun Slack bot that renders 3D ASCII shapes, giant text, and art scenes — all from a simple slash command. No notifications. No alerts. Just cool stuff in your chat.

SLASH COMMANDS

7 commands. Zero fluff.

Every command is a slash command inside Slack. Type it, get something back. That's the whole deal.

Draws a 3D ASCII shape — cube, pyramid, prism, cylinder, diamond, or cone — sized to your spec. Built with pure character math, no libraries.

HOW IT WORKS

Simple by design.

JARVIS runs in socket mode — no public server, no webhooks. Just a persistent connection and a handler for each command.

01

You type a slash command

Type /jarvis-shape, /jarvis-text, or any command directly into any Slack channel or DM. Add your inputs on the next lines.

02

Slack sends it to the bot via socket

The Slack Bolt framework receives the command payload over a persistent WebSocket connection. No public URL required.

03

The handler generates the output

For shapes, pure JS math builds each row of characters. For text, a hand-coded A–Z glyph map assembles the letters. For fun art, pre-built scenes are fetched from a local library.

04

The response appears in Slack

The bot responds with a formatted message wrapped in a code block. Monospace rendering preserves every character's alignment — the art looks exactly right.
